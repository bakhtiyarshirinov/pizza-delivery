Pizza Delivery App
A simple and efficient web application for ordering pizza online. This project covers the full flow from browsing the menu to placing an order.

Features
Menu: Browse various pizzas with prices and descriptions.

Cart System: Add, remove, and update item quantities.

Checkout: Simple form to submit delivery details.

Responsive: Works on desktop, tablets, and mobile phones. 📱

Tech Stack
Frontend: React.js / NextJS / Tailwindcss

Backend: Node.js / Express

Database: MongoDB (or your choice of DB)

State Management: Redux or Context API
